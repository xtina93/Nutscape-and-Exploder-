Nutscape and Exploder Challenge

Background

This assignment is an homage to the earliest days of the commercial web. Back in the mid-1990s when the browser wars were heating up, developers had their own pet names for Netscape Navigator ("Nutscape") and Internet Explorer ("Internet Exploder") that reflected just how frustrated they were with the state of the art. These best available browsers were horribly unstable, crashing multiple times per day (or hour). To make matters worse, both Netscape and Microsoft tried to stay ahead of the standards by adding proprietary language features and tags, making it impossible sometimes for developers to get their pages working in both browsers at the same time.

The Challenge

In this folder are two html files, netscape.html and microsoft.html, each a lightly edited version of the homepage of each company in the mid-1990s. Neither is anywhere near valid in any recent version of HTML. Your task is to revise each page so that it follows contemporary standards and norms in in HTML5 and CSS3. When complete, the pages should replicate the content and look of the pages as closely as possible using valid HTML5 and CSS3.

There are a few rules to follow when making your revisions:

  1. Please do not edit the original files. Instead, edit the copies with "-html5" in the filename.
  2. If you delete an obsolete or deprecated element then replicate the intended effect of the original markup using HTML5/CSS3 equivalents.
  3. Do not delete any text content.
  4. Logos and other meaningful images should be retained. (They don't belong to us!) Spacer gifs and other non-meaningful images should be deleted.
  5. Do not reorder any tags or content unless absolutely required to comply with HTML5 standards.
  6. Table layouts and other elements that are being used in a non-semantic fashion are to be replaced with equivalents that meet HTML5 norms.

Rather than attempt to complete this challenge in one sitting, we will instead approach it in small installments that reflect where we are in the textbook.
